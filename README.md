# TaskCampus

Sistema web minimalista para la gestión de tareas académicas, desarrollado bajo la metodología Spec Driven Development (SDD).

## Descripción del Proyecto
TaskCampus permite a los estudiantes registrar, consultar, actualizar, eliminar y filtrar tareas. El proyecto está dividido en un frontend construido con Vite, TypeScript y TailwindCSS, y un backend desarrollado con Python y FastAPI, utilizando persistencia de datos en un archivo JSON.
